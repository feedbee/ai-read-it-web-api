# AI Read It Web API

## Description

This project is a Web API that uses AI to read.

## Installation

```bash
# Clone the repository
git clone https://github.com/feedbee/ai-read-it-web-api.git

# Navigate into the directory
cd ai-read-it-web-api

# Install dependencies
npm install
```

# Usage

```bash
# Start the server
npm start
```
